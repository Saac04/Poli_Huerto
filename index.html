ADECUA TU ESPACIO

Sensor de humedad para jardines que mide la humedad del suelo y ayuda a determinar cuándo es necesario regar las plantas para evitar que se sequen o se ahoguen.

Sensor de salinidad mide la cantidad de sal disuelta en el agua y ayuda a determinar si es adecuada para el cultivo de las plantas.

El sensor de temperatura mide la temperatura de un entorno y ayuda a determinar si es adecuado para el crecimiento de las plantas.

El sensor de pH mide el nivel de acidez o alcalinidad de una solución y ayuda a determinar si es adecuada para las plantas, ajustando los niveles de pH si es necesario.

Sensor de luz que determina si la planta está recibiendo suficiente luz para su crecimiento y si es necesario ajustar la ubicación o la fuente de luz.
